## 🌐 Deploy no Vercel

1. **Criar repositório GitHub**:
   ```bash
   git init
   git add .
   git commit -m "Portfolio inicial"
   git remote add origin https://github.com/SEU_USUARIO/portfolio
   git push -u origin main
   ```

2. **Conectar no Vercel**:
   - Acesse vercel.com
   - Conecte seu GitHub
   - Deploy automático!
